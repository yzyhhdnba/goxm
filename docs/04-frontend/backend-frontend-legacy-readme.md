# Backend Frontend Legacy Notes

该文档来自 `backend/frontend/` 历史副本，仅用于保留原始前端工程的本地命令说明。

当前工作区默认以前端主目录 `frontend/` 为准，除非明确需要核对历史副本，否则不要继续以 `backend/frontend/` 作为主要开发入口。

## Project Setup
```
yarn install
```

### Development
```
yarn serve
```

### Production Build
```
yarn build
```
