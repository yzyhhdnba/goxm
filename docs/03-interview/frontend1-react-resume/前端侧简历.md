# PILIPILI Go 并行 React 前端重构项目

时间：2026.04 - 至今

项目介绍：

在保留原有 Vue 业务前端的基础上，新建 `frontend1/` 作为并行 React 前端工程，围绕在线视频平台核心链路完成首页推荐、搜索、视频详情、评论回复、个人中心、历史记录、创作者投稿、审核后台与系统通知等页面重构，并接入 Go 后端真实接口，形成可独立构建和联调的新前端实现。

主要技术栈：

- React
- TypeScript
- Vite
- Tailwind CSS
- Redux Toolkit
- React Router
- Axios

主要工作：

- 使用 Vite 搭建 React 前端工程，结合 TypeScript、Redux Toolkit 与 React Hooks 完成项目初始化、全局状态管理与页面状态组织
- 基于 Axios 封装统一请求层，通过请求拦截器与响应拦截器处理 token 注入、错误收口和数据解包，提升接口复用性
- 按业务域拆分首页、搜索、视频详情、评论树、个人中心、历史记录、投稿、审核和通知模块，抽离可复用组件与布局骨架
- 实现视频详情页的评论与回复树交互，支持一级评论、二级回复展开、发布回复、评论点赞和回复点赞等功能
- 实现搜索页、用户主页与视频详情之间的来源页透传和分页条件保留，保证多页面跳转下的上下文状态不丢失
- 实现个人仪表盘、播放历史、创作者投稿页、管理员审核页和系统通知页，补齐平台用户侧与管理侧的核心前端闭环
- 基于浏览器原生 `video + canvas` 实现视频封面截帧能力，并持续通过构建验证保证并行 React 前端的稳定交付
