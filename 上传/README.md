# 上传目录

`上传/` 用于存放适合上传到“面试项目知识库”的文档副本。

这里的文件不是原始事实来源，而是从根目录 `README.md` 和 `docs/` 中挑选并同步出来的上传版本。

## 使用原则

- 如果原始文档有更新，需要重新同步到这里。
- 这里优先保留高信息密度、适合问答检索的文档。
- 低信息密度的本地脚手架说明、历史遗留说明，默认不放进这里。
